# .sailsrc

This file is useful for setting configuration that you want to generate programmatically.  You can also use it to extend the functionality of the Sails CLI tool, for example to add a generator.

You can learn more about using sailsrc files [here](https://sailsjs.com/documentation/concepts/configuration/using-sailsrc-files).
